# Список запрещённых слов и словосочетаний

- `<…>-модуль` → `модуль <…>`
- `API-<…>` → `<…> API`
- `eval-кейс` → `проверочный пример`, `сценарий оценки`, `случай проверки`
- `entrypoint` → `точка входа`
- `Git-репозиторий` → `репозиторий Git`
- `HTTP-<…>` → `<…> HTTP`
- `OAuth-<…>` → `<…> OAuth`
- `judge` → `проверка`, `сравнение`
- `practically` → `на практике`, `на деле`, `почти`, `по сути`
- `oracle` → `эталон`, `эталонный результат`, `эталонная проверка`
- `read-only` → `только для чтения`, `без записи`
- `runner` → `запускатель`, `средство запуска`
- `source-of-truth` → `источник истины`
- `TLS-<…>` → `<…> TLS`
- `VPN-<…>` → `<…> VPN`
- `{…}-файл` → `файл {…}`
- `апдейт` → `обновление`
- `баг` → `ошибка`
- `батч` → `набор`, `пакет`, `пакетный запуск`
- `батчем` → `скопом`, `разом`
- `дока` → `документ`, `документация`, `описание`, `справка`
- `доки` → `документация`, `документы`, `описания`
- `ИИ-модель` → `модель ИИ`
- `кастомный` → `пользовательский`, `настраиваемый`, `особый`, `настроенный`
- `код-ревью` → `обзор кода`
- `конфиг` → `настройки`, `конфигурация`
- `лог` → `журнал`
- `маппинг` → `сопоставление`, `отображение`
- `нативный`
- `онбординг`
- `пайплайн` → `сборочная линия`, `конвейер`
- `ревью` → `обзор`, `проверка`
- `ретрай` → `повтор`, `повторная попытка`
- `снапшот` → `снимок`
- `топ-{N}`
- `трекинг времени` → `учёт времени`
- `фича` → `возможность`
- `чек-лист` → `проверочный список`
- `шарить` → `делиться`
