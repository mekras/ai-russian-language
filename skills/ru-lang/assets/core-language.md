# Общий языковой слой

## Общие правила

- Используй букву «ё».
- Для русских кавычек используй только «…».
- Не используй английские типографские кавычки `“` и `”`, а также не заменяй
  ими русские кавычки `«` и `»`.
- Перед отправкой любого русского текста быстро проверь его на наличие `“` и
  `”`.
- Перед отправкой любого русского текста, включая промежуточные комментарии и
  отчёты о работе, быстро проверь его по разделам ниже.

## Базовая логика проверки

Следи за однотипными англо-русскими кальками и заменами не только по отдельным
словам из списков, а по повторяющимся конструкциям.

- Применяй замену в соответствии с закреплёнными контекстами проекта.
- Не изменяй цитаты, URL, команды, имена файлов, веток, пакетов, классов,
  методов, параметров и другие технические идентификаторы, если это не задача
  редактурной правки.
