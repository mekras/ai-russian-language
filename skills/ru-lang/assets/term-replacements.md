# Словарь замен

Используй этот файл как справочник конкретных замен. Он не отменяет проверку
по смыслу и не заменяет правила из `references/`.

- `корректный` → `правильный`, `верный`
- `корректен`, `корректна`, `корректно`, `корректны` → `правильный`,

  `без синтаксических ошибок`
