# Навыки использования русского языка для ИИ

Кооллекция переносимых навыков для агентов ИИ, которые помогают им правильно
применять русский язык.


## Навыки

- `ru-dev` — русский текст в разработке ПО.
- `ru-lang` — основные правила русского текста.


## Установка и обновление

Основной способ использования — установка коллекции через
[Agent Package Manager](https://github.com/microsoft/apm).

Установить всю коллекцию:

```bash
apm install --global github.com/mekras/ai-russian-language#^1.0.0
```

Установить только выбранные навыки:

```bash
apm install --global github.com/mekras/ai-russian-language#master --skill ru-lang
```

Обновить уже установленную коллекцию:

```bash
apm deps update --global
```

Проверить, есть ли новые версии зависимостей:

```bash
apm outdated --global
```

## Использование

После установки навыки становятся доступны агенту как обычные локальные
навыки. Дальше их можно вызывать явно по имени или использовать неявное
срабатывание там, где оно поддерживается средой.

## Разработка

Запуск проверок:

```bash
apm run tests
```

Публикуемые навыки лежат в `skills/<name>/`.
При добавлении или изменении навыка:

1. Обновите `SKILL.md`.
2. Синхронизируйте `agents/openai.yaml`, если меняется назначение навыка.
3. Добавьте или обновите `evals/triggers.json`.
4. При заметных изменениях обновите [CHANGELOG.md](./CHANGELOG.md).
