# История изменений

Все заметные изменения в этой коллекции навыков отражаются в этом файле.

Формат основан на `Keep a Changelog`, а версии следуют `SemVer`.

## [Не выпущено]

## [1.2.1] - 2026-06-18

### Изменено

- Упрощён `ru-lang`: длинный список замен вынесен в справочники, а старые
  дублирующие файлы удалены из публикуемого состава.

### Исправлено

- Обновлён `apm.lock.yaml` после удаления устаревших файлов и повторной
  установки локального состава.

## [1.2.0] - 2026-06-18

### Изменено

- Исходники навыков перенесены в `.apm/skills/`, а `apm run tests` переведён
  на структурные проверки и выборочный запуск через `APM_EVAL_PATH` и
  `APM_EVAL_CASE_ID`.

### Исправлено

- Стабилизированы сценарии результата `ru-lang` для гибридных форм с DNS,
  `single-file`, Claude и формами с корнем `корректн`.

## [1.1.2] - 2026-06-18

### Исправлено

- Усилены правила `ru-lang` для гибридных форм из отчётов агента с `APM`,
  `Claude`, `diff` и `lockfile`.
- Добавлен регрессионный сценарий `ru-lang` для отчёта по сессии
  `019ed59e-6610-7472-a22f-fc1c676cfd96`.

## [1.1.1] - 2026-06-17

### Добавлено

- Проверки срабатывания `ru-dev` для коротких запросов на коммит.

### Исправлено

- Усилены правила `ru-lang` для русских названий вместо slug-форм и для буквы
  «ё» при правке русских фраз.
- Стабилизирована составная проверка `ru-lang` для естественных форм
  словосочетания с `Inspect`.
- Исправлен запуск подагентов проверки для текущих значений
  `approvals_reviewer` в Codex CLI.

## [1.1.0] - 2026-06-15

### Изменено

- Рефакторинг, улучшения проверок и тесты.

## [1.0.0] - 2026-06-08
